Covid-19 Trends in South Africa

Effective Reproduction Number

COVID-19 SA Goverment Portal

To find out how fast Covid-19 is spreading, we can calculate the Effective Reproduction Number (Rt). This number tells us, on average, how many people each infected person infects.
While Rt is above 1, the virus is spreading fast. Getting it below 1 is key to stop the spread.
This website provides up to date Rt charts for the country, provinces, districts and subdistrict (were data is available). Only areas with significant spreading will be shown.
